Command Line Parameters

Use parameters to customize your game startup.

"-s random" will run the base Wordle game.

"-s absurdle" will run the Absurdle game.

No parameters will default to the base Wordle game.

Game Options

  • Random: The game selects one secret target word at the start of the session from the dictionary.
  • Absurdle: The game does not pick a single word at the start. Instead, it shifts the target word based on your guesses to make the game as difficult as possible.
